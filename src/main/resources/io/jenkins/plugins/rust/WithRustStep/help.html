Configures a Rust environment for the wrapped block.

This step sets up the Rust environment (CARGO_HOME, RUSTUP_HOME, PATH) for all commands executed within its closure.

Examples:

Scripted Pipeline:

withRust(rustInstallationName: 'stable') {
    sh 'cargo build --release'
    sh 'cargo test'
}

Declarative Pipeline:

pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                withRust(rustInstallationName: 'nightly') {
                    sh 'cargo build'
                }
            }
        }
    }
}

Note: For simpler usage, consider using the tools block or the dedicated pipeline steps like cargoBuild(), cargoTest(), etc.
