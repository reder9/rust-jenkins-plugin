Runs a Cargo command in your Rust project.

Examples:

Basic usage:

cargo(command: 'build')

With arguments:

cargo(command: 'build', args: '--release')

With specific Rust installation:

cargo(command: 'test', rustInstallationName: 'Rust-Stable')

Run clippy:

cargo(command: 'clippy', args: '--all-targets -- -D warnings')

If no Rust installation is specified, the step will use the Rust installation configured in the tools block, or the first available installation, or system cargo if none is configured.
